Connect once. Your terminals stay within reach.

Connect your devices

Install Pedals on your computer and iPhone. Open Connect on the computer, then enter its 8-digit code on iPhone before the code expires. Keep the desktop app running in the menu bar while using remote terminals.

If a connection is unavailable

  • Confirm both devices can reach the internet.
  • Open the Pedals menu bar app and check that it says Connected.
  • Create a fresh connection code if the previous code expired.
  • Install the latest desktop release and iPhone build.

Contact

Email eyhn@yellowplus.app with your device model, OS version, and a short description of the issue. Do not include terminal contents or pairing codes.
